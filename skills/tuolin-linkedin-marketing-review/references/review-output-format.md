# Marketing Review Output Format

Write `01_中文策划_营销审阅.md` with this structure:

```markdown
# LinkedIn 中文策划营销审阅

## 审阅结论

- 结论：建议通过 / 建议修改后通过 / 不建议直接继续
- 是否建议进入中文 30 天贴文总稿：是 / 否

## 审阅范围

- 只审阅 `01_中文策划.md` 的营销质量、市场适配和对外表达风险。
- 不调用知识库复核项流程。
- 不写回正式知识层。

## 检查项

- 欧美 B2B 市场适配：...
- 对外中文名：...
- 对外英文名：...
- 核心卖点：...
- 发布方式：...

## 风险

- ...

## 修改建议

- ...

## 下一步

- 如采纳：生成中文 30 天贴文总稿时应用上述建议。
- 如不采纳：仍可继续生成中文 30 天贴文总稿，但 manifest 和总稿中应记录未采纳状态。
```

Keep recommendations concrete enough for the next generation step to apply.
