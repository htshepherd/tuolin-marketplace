# Style Prompt Rules

Build the image prompt from:

- selected source image
- Day theme
- Day post angle
- Day hashtags
- Day visual tags
- selected style category
- transparent Tuolin logo
- contact email

## Required English Text

Use concise English labels. Prefer:

- `TUOLIN Specialty Glass Fiber Tape`
- `1000°C Working Temperature`
- `Itch-free Handling`
- `No Smoke`
- `tuolin@tuolintech.com`

## Visual Rules

- Keep the product recognizable.
- Use the original image dimensions by default.
- Keep logo background transparent; never place the logo inside an opaque white card unless the category explicitly calls for a brand block and the user approves.
- Tags should feel integrated into the layout, not pasted-on stickers.
- Avoid heavy dark masks over white product photos unless the category is explicitly dark industrial.
- Leave enough clear space around the product.
- Do not cover important product texture, roll holes, pipe joints, or installation details.

## Output

Save a single image file named with Day and category, for example:

`day-01-original-light-enhancement.png`
