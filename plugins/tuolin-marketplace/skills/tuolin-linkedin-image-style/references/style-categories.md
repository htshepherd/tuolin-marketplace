# LinkedIn Publishing Image Style Categories

Use only one selected category per generation.

1. `original-light-enhancement` / 原图轻量增强型: Add transparent logo, tags, and email with minimal visual change.
2. `minimal-premium` / 极简高端型: Use restrained spacing, clean type, and premium product focus.
3. `industrial-technical` / 工业技术型: Add fine-line callouts and technical labels.
4. `engineering-drawing` / 工程图纸型: Use dimension-line or blueprint-inspired annotations without inventing dimensions.
5. `product-detail-closeup` / 产品细节特写型: Emphasize weave, edge, thickness, and tactile quality.
6. `technical-parameter-card` / 技术参数卡片型: Present `1000°C`, `Itch-free`, and `No smoke` as clear parameter blocks.
7. `three-benefit-banner` / 三大卖点横幅型: Make the three core benefits the dominant message.
8. `application-scenario` / 应用场景型: Use only when the source image supports a real application scene or the user accepts AI scene creation.
9. `exhaust-wrap-scenario` / 排气管包覆场景型: Use for exhaust-focused content and source images.
10. `industrial-pipe-insulation` / 工业管道保温型: Use for pipe, equipment, or factory insulation themes.
11. `high-temperature-test` / 高温测试型: Use heat visuals carefully; do not fabricate certification or exact test proof.
12. `cutting-processing` / 切割加工型: Emphasize cutting or installation; avoid smoke visuals because the claim is no smoke.
13. `comfortable-handling` / 手感舒适型: Emphasize touch and installer comfort.
14. `pain-point-solution` / 痛点解决型: Frame itchy handling, smoky wraps, or messy installation as buyer pain points without attacking competitors.
15. `buyer-checklist` / 采购清单型: Use checklist layout for B2B buyers.
16. `inquiry-conversion` / 询盘转化型: Make email and CTA prominent.
17. `specification-display` / 规格展示型: Show width, thickness, roll length, or customization only when supplied by the Day content or user.
18. `multi-spec-combination` / 多规格组合型: Use only with real multi-spec source material or explicit AI-generation permission.
19. `split-screen-comparison` / 分屏对比型: Compare pain point vs solution without naming or attacking competitors.
20. `qa-education` / 问答科普型: Lead with one question tied to the Day topic.
