# Size Recommendations

Default: preserve the source image dimensions.

Suggest a different size only when the user asks or the source image is unsuitable for LinkedIn.

Recommended LinkedIn-friendly alternatives:

- Square feed: `1080x1080`
- Portrait feed: `1080x1350`
- Landscape feed: `1200x627`

When resizing is approved:

- Do not distort the product.
- Preserve aspect ratio through crop or padding.
- Prefer clean margins over cutting off the product.
