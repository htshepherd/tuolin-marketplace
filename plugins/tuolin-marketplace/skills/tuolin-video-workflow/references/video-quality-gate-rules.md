# 视频创作交付门禁

本文件保留为兼容引用。当前视频 Agent 的门禁围绕策划、分镜、SRT、即梦任务和镜头文件完整性，不负责最终音频或发布。

## 策划门禁

- 已通过正式 Agent 接口读取产品知识卡和官方图片。
- 受众、问题场景、观看动机和最终观众兴趣方向已确认。
- YouTube 趋势扫描有来源和日期，或明确记录无法完成实时扫描的降级状态。
- 候选图已逐张打开并检查主体、清晰度、构图、9:16 裁切和近重复。
- 策划事实不超出知识卡，方向可由现有素材或明确批准的安全 AI 模拟执行。

## 分镜与 SRT 门禁

- 镜头数、时长和角色根据当前策划动态设计，不使用固定五秒或固定角色序列。
- 每个镜头显示用途、动作、有序参考图、连续性、风险及对应 SRT 或有意静默。
- `storyboard.srt` 是唯一未来旁白逐字稿，语义必须与镜头或连续镜头组一致。
- Prompt 只能在分镜与 SRT 共同确认后生成。
- 多图镜头必须有 provider 能力证明；否则回到分镜拆成相邻镜头并重新确认。

## 即梦与合并门禁

- 任务计划显示镜头映射、图片、时长、额度、适配和 blocker。
- 付费授权使用 `确认即梦生成`，真实提交另行确认。
- dry-run 不能当作真实结果。
- 用户在网页端确认全部生成后，只允许一次真实 task ID 下载；任务数、镜头映射、文件存在性、可读性和非空都必须通过。
- 合并严格使用已确认分镜顺序，并保留 `storyboard.srt`；不生成冲突的旁白或字幕 Markdown。
- 结果不接受时停止当前运行。本期不轮询、不静默重试、不单镜头重做。
